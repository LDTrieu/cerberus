# Cerberus

Cerberus is a library to provide common codes for backend dev

## Installation
- Please make sure export `GOPRIVATE`

```bash
export GOPRIVATE=github.com/ldtrieu/cerberus/*
go get github.com/ldtrieu/cerberus
```

## Run test
Run tests

```bash
make tests
```

Run lint

```bash
make lint
```

Clean

```bash
make clean
```
